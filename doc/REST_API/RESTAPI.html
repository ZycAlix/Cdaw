Root Endpoint

The root endpoint should prefix all resources and is only accessible over HTTPS.

http://localhost:8888/projet-cdaw/BackEnd/src/api.php

Get Utilisateurs

Method: GET

Get informations of all users from API.

http://localhost:8888/projet-cdaw/BackEnd/src/api.php/users

Get a Single User

Method: GET

Get informations of the user whose id is 1.

http://localhost:8888/projet-cdaw/BackEnd/src/api.php/user/1
